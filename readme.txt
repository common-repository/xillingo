== Installation ==

= Requirements / Requisitos de instalación=
* PHP 5.2.4 or higher.
* WordPress v3.0.x or higher.
* *You shall register with Xillingo. The account is 100% free.*

= Install Via Plugin Repository =
1. Log into WordPress admin for your site
2. Go to 'Plugins -> Add New' and choose 'Search'
3. Search for Xillingo and click 'Install Now'
4. Activate the plugin through the 'Plugins' menu
5. Go to ‘Xillingo -> Activación’ and click 'Register your plugin'
6. Enjoy!

== Frequently Asked Questions ==

= How quickly does it start working?  / ¿Que tan rápido empieza a funcionar?=

The Xillingo Widget will start to work immediately; however, part of the optimization process involves learning which of your content performs the best. A certain number of impressions must be served before the widget is able to fully optimize and display the most effective content.

El Widget de Xillingo va empezar a funcionar inmediatamente: Sin embargo, parte del proceso de optimización requiere aprender de tu contenido para que rinda lo mejor posible.

= Does Xillingo work on web sites in any language? / ¿Funciona Xillingo en sitios de cualquier lenguaje?=

Currently, the tool will only work properly on Spanish language websites, but we are working pretty much for including english websites.

Actualmente, el Widget está optimizado para sitios en español, pero estamos trabajando arduamente para incluir sitios en inglés. 

= Will the widget slow down my site? / ¿Xillingo hará que mi sitio sea más lento?=

No. The Xillingo widget is designed to load _"asynchronously"_. This means your page will load independently of the widget script.

No. El Widget de Xillingo está diseñado para cargar “Asíncronamente”. Esto significa que tu sitio carga de manera independiente al widget.

= I have sponsored content on my site with clicks but don’t see any revenue. / Muchos clicks pero nada de ganancia =

Revenue is based on actions not clicks.  Sponsored content served through the system won't generate revenue unless the user performs a particular action on the advertisers site _(ex. Signs up for a service or purchases something)_. When this happens the payout is generally much better than what you expect to see from a pay-per-click advertisement. This means that you might need to let the widget run for a while to start seeing revenue.

Las ganancias son basadas por acciones, no por click. El Widget va a generar dinero siempre y cuando el usuario realice una acción, por ejemplo una compra.
